Análise de Ciclos de Crédito: Correlação Selic vs. Inadimplência (2012-2026)

Descrição do Projeto:
Este estudo analisa a dinâmica entre o custo de capital (Selic) e o risco sistêmico (Inadimplência) no mercado brasileiro. Utilizei um framework proprietário em Python para garantir a sincronia temporal das séries históricas extraídas do Banco Central (SGS), permitindo identificar o lag (atraso) de resposta do mercado de crédito aos ciclos monetários.

Principais Insights
Sincronia Temporal: O gráfico demonstra como os picos de Selic (Linha Vermelha - Eixo Direito) precedem ou acompanham as elevações na taxa de inadimplência (Linha Azul - Eixo Esquerdo).
Janela 2024-2026: Observa-se a manutenção de juros elevados e a pressão contínua sobre a capacidade de pagamento, um cenário crítico para empresas intensivas em capital.

Metodologia: Os dados foram tratados para neutralizar ruídos sazonais, garantindo que as tendências observadas reflitam o comportamento real do ciclo de crédito.
